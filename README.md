# 心流学社 Dashboard 原型

这是一个静态网页原型，用于展示面向中国学生的时间、生活与知识管理平台概念。仓库包含：

- `index.html`：页面结构与文案
- `styles.css`：视觉样式

## 本地预览

1. 在项目根目录启动简单的静态服务器：
   ```bash
   python3 -m http.server 8000
   ```
2. 浏览器访问 [http://localhost:8000/index.html](http://localhost:8000/index.html)。

也可以直接双击 `index.html` 在浏览器中打开，浏览器会自动加载同目录下的样式文件。
